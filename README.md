# 🎓 Colégio Técnico KNOW

🔗 **Acesse o projeto online:**
https://larahamatos.github.io/projetoFinal.ProWEbI/

Um site institucional moderno e interativo desenvolvido para o **Colégio Técnico KNOW**, com foco na apresentação dos cursos, captação de alunos e experiência dinâmica para futuros estudantes.

O projeto conta com um design responsivo, navegação SPA (Single Page Application), quiz vocacional inteligente, raspadinha premiada e sistema de pré-cadastro de alunos.

---

## 🚀 Funcionalidades

### 🏠 Página Inicial

* Apresentação institucional do Colégio Técnico KNOW
* Destaque dos cursos oferecidos
* Estatísticas e indicadores da instituição
* Chamadas para ação (CTA)

### 📖 Página Sobre

* História da instituição
* Linha do tempo interativa
* Missão, visão e valores
* Indicadores de desempenho

### 🎓 Cursos Técnicos

* Técnico em Informática
* Técnico em Enfermagem
* Técnico em Administração
* Informações detalhadas sobre:

  * Duração
  * Carga horária
  * Módulos
  * Certificações
  * Mensalidades

### 🎯 Quiz Vocacional

* Seleção aleatória de perguntas
* Avaliação de perfil profissional
* Resultado personalizado
* Recomendações de curso com base nas respostas

### 🎰 Raspadinha Interativa

* Sistema de raspagem utilizando Canvas
* Prêmios aleatórios
* Controle de tentativas via LocalStorage
* Experiência gamificada para captação de alunos

### 📝 Pré-Cadastro

* Cadastro de interessados
* Armazenamento local dos dados
* Exibição dinâmica dos registros

### 📱 Responsividade

* Compatível com:

  * Desktop
  * Tablet
  * Smartphone

---

## 🛠️ Tecnologias Utilizadas

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
* **LocalStorage**
* **Canvas API**
